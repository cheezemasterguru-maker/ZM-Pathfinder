ZM Pathfinder

Static single-file app for GitHub Pages. Grid is always 7 columns wide. Cost priority is exponential first, tile count second.

Grid input

Use 7 comma-separated cells per row. Allowed values: numbers, X, B, S, . for empty. Example: 1,2,.,.,.,3,.

Rules hardcoded

• Red = gate priority only. Bubble only if it naturally sits on red's optimal gate route.
• Blue = shaft priority, solved independently first. Blue reuses red only when cheaper.
• All bubbles are mandatory overall.
• Movement is orthogonal only.
• Shafts are touch-only and rendered as merged 2×3 blocks.
• Empty cells are free traversal. Black cells are blocked.
• Normal gates and graveyards use 5 attack points. End gates use 3.
