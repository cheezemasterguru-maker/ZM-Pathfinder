ZM Pathfinder

Paste from Google Sheets / Excel or build visually. Grid is always 7 wide. Cost priority is exponential first, tile count second.

Paste from Sheets / Excel

Supported values: numbers, B, Shaft, X / Block / ■, blank cells. Mobile copy sometimes strips tabs, so this importer now auto-pads rows and also supports loose pasted text.

Visual placement tools

Tap a tool, then tap cells in the preview board. Shaft fills a full 2×3 block starting from the tapped cell as the top-left corner.

Rules hardcoded

• Red = gate priority only. Bubble only if it naturally sits on red’s optimal gate path.
• Blue = shaft priority, solved independently first. Blue reuses red only when cheaper.
• All bubbles are mandatory overall.
• Movement is orthogonal only.
• Shafts are touch-only and rendered as merged 2×3 blocks.
• Empty cells are free traversal. Black cells are blocked.
• Normal gates and graveyards use 5 attack points. End gates use 3.
