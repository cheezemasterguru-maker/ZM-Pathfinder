ZM Pathfinder

How input works
1. Tap a cell in the board.
2. Copy a number range from Sheets / Excel.
3. Tap Paste From Clipboard or use Paste Into Board from the box.
4. Use the visual tools for Block, Bubble, Shaft, and Empty.
Visual placement tools
Layout restored. Grid paste is wired in. Solver not wired into this merged layout yet.
Editable Board
Tap a cell to select it. Paste starts from that cell. Tap special tiles again to remove them.
