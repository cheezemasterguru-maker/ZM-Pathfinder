ZM Pathfinder

How input works
1. Tap a cell in the board.
2. Copy a number range from Sheets / Excel.
3. Tap Paste From Clipboard to place it starting from the selected cell.
4. Use the visual tools for Block, Bubble, and Shaft.
2-file version loaded. Grid editor and solver are separated.
Editable Board
Tap a cell to select it. Paste starts from that cell. Tap special tiles again to remove them.
