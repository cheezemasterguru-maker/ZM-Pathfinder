ZM Pathfinder

Fixed 7×20 board. Paste number ranges directly into the visible grid starting from the selected cell. Place blocks, bubbles, and shafts visually.

How input works

1. Tap a cell in the board.
2. Copy a number range from Sheets / Excel.
3. Paste with Ctrl/Cmd+V on desktop or your phone’s paste command.
4. Use the visual tools for Block, Bubble, Shaft, and Empty.

Visual placement tools

Rules hardcoded

• Board is always 7 columns × 20 rows.
• Paths start one blank row below the lowest used map cell.
• Red = gate priority only. Bubble only if naturally on red’s optimal path.
• Blue = shaft priority, solved independently first. Blue reuses red only when cheaper.
• All bubbles are mandatory overall.
• Movement is orthogonal only.
• Shafts are touch-only and rendered as merged 2×3 blocks.
• Empty cells are free traversal. Black cells are blocked.
• Normal gates and graveyards use 5 attack points. End gates use 3.

Editable Board

Tap to select a cell. Paste numbers into the board from the selected cell. Special tiles are placed visually.
