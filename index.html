ZM Pathfinder

How input works

Tap a cell in the grid. Then either type a number directly into that cell or paste copied Sheet/Excel ranges directly into the grid starting from the selected cell. Use the tools above the grid for Block, Bubble, Shaft, and Empty. Tap the same special tile again to remove it.

Global solver rules

• Exponential weighting is global for all numbered cells.
• Movement is orthogonal only.
• Standard vs End only changes attack width, not weighting logic.
• Red = gate objective.
• Blue = shaft objective and may reuse red where that lowers weighted cost.
• Bubbles are assigned by lower weighted total route cost.
• Paths start one blank row below the lowest used map cell.
• Empty cells are free. Black cells block. Shafts are touch-only 2×3.

Editable Board

Direct paste works in the grid itself. Tap a cell first, then paste.
