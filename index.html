ZM Pathfinder

How input works

Tap a grid cell first. Paste copied number ranges into the visible paste box on the right, then tap Apply Paste. Use the visual tools for Block, Bubble, Shaft, and Empty. Tap the same special tile again to remove it.

Visual placement tools

Rules hardcoded

• Board is always 7 columns × 20 rows.
• Paths start one blank row below the lowest used map cell.
• Movement is orthogonal only.
• Normal gates / graveyards use 5 attack points. End gates use 3.
• Blue may reuse red path at zero added weighted cost.
• Empty cells are free. Black cells are blocked. Shafts are touch-only 2×3 regions.
• Bubble ownership is chosen by cheaper total weighted route, but this version is still a heuristic and may still need tuning on edge cases.
